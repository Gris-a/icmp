# icmp/dns

## Usage
```
    g++ main.cpp
    ./a.out <IP> <INTERFACE1> <INTERFACE2>
```
```
    traceroute -I <IP>
```
IP - ip адресс, на котором будет вызываться traceroute. INTERFACE1 - интерфейс, подключённый к устройству с которого вызывается traceroute, INTERACE2 - другой интерфейс, подключенный к сети.
